# World Countries Explorer

This project is a web application that lists all countries in the world using data from the **REST Countries API**.

The application allows users to search for countries and view detailed information such as population, capital, region, subregion, and geographic coordinates. When a country is selected, a modal is displayed with additional details and an interactive map showing the countryâ€™s location.

The map functionality is implemented using **Leaflet**, a JavaScript library for interactive maps, which uses the latitude and longitude provided by the API.

This project was built with **React** and focuses on practicing API consumption, component-based architecture, and UI organization.

---

## Features

- List all countries in the world  
- Search countries by name  
- View country details in a modal  
- Display country location on an interactive map  
- Responsive layout  

---

## Technologies

- React  
- REST Countries API  
- Leaflet.js  
- JavaScript (ES6+)  
- CSS Modules  
- Bootstrap

## How to Clone and Run

### 1.Clone the repository
```bash

```

### 2. Enter the project folder
```bash
  cd world-countries-explorer
```

### 3. Install dependencies

```bash
 npm install

```

